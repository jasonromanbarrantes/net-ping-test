15-Minute Ping Test

This runs a local ICMP ping for 15 minutes to LanguageLine targets and saves results on your PC.

1) Download the script

Download RunPingTest_NoServer.ps1

If Windows flags the file, right-click → Properties → check Unblock.

2) Run it (PowerShell as Admin)

Open PowerShell as Administrator and allow scripts for this session only:

Set-ExecutionPolicy -Scope Process Bypass -Force

Then run (IC = Interpreter Connect, Linc = Linc users):

cd <folder where you saved>
.\RunPingTest_NoServer.ps1 -TestType IC -DurationMinutes 15

Outputs

  • Folder in your %TEMP% with raw ping text files for each IP
  • summary.json with packet loss and min/avg/max
  • ZIP archive of all artifacts

You can attach the ZIP or paste the last lines of the raw files into an email/ticket.

Browsers can’t ping; that’s why this uses a PowerShell script.
